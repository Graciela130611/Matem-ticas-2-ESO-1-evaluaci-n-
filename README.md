# Matemáticas 2º ESO - 1ª evaluación

Material de apoyo para la recuperación de Matemáticas de 2º ESO con ejercicios guiados y ejemplos.

## 📚 Contenidos
- Números enteros
- Operaciones básicas
- Problemas aplicados
- Ejercicios de refuerzo

## 🎯 Uso educativo
Este recurso está pensado para el alumnado de Educación Secundaria Obligatoria como material de apoyo y refuerzo.

Puede ser utilizado tanto en el aula como en casa.
